# SKARA Atelier Shoes

Luxury multi-page static storefront for a premium shoe boutique.

## Pages

- `index.html` - Home, hero slider, brand marquee, category rooms, featured product rail
- `collections.html` - All products with search, category, audience, and sort filters
- `men.html` - Men's edit
- `women.html` - Women's edit
- `kids.html` - Kids' school, sport, and weekend edit
- `concierge.html` - Client-side concierge request flow

## Run

Open `index.html` directly in a browser, or run a local static server:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:4173/index.html
```
